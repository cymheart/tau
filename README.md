# tau Midi数字音乐合成器库(tau midi synthesizer lib)
该库具备Midi音乐解析播放，soundfont解析播放，支持混音，合唱，压缩器，均衡器等音效添加。
支持边缓存边播放模式
支持对黑乐谱的流畅播放。
支持C#播放midi音乐等


### 支持系统平台
目前支持
   Window， 
   Android 平台
   
### 工程文件包括测试示例
TauClrTest(C#)
TauTest(C++)

### Visual Studio 编译
VS工程启动 直接运行编译

### Android Studio 编译
AS工程启动 直接运行编译
