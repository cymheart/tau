# tau Midi数字音乐合成器库(tau midi synthesizer lib)
该库是作者自研的数字音乐合成器库， 目前正处于完善发展中，您的支持将是这个库不断进步的动力。
该库具备Midi音乐解析播放，soundfont解析播放，支持混音，合唱，压缩器，均衡器等音效添加。
支持对黑乐谱的流畅播放。
支持C#播放midi音乐等

### tau实现的目标
该库最终的目标将是实现一个全面的高性能的数字音乐工作站支持库，
将会是一段漫长的路途，您的支持将是这个库以及作者不断进步的动力。
