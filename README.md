# tau Midi数字音乐合成器库
1.一个数字音乐合成库，该库具备soundfont解析，Midi音乐解析播放，支持混音，合唱，压缩器，均衡器等音效添加  
2.支持边缓存边播放模式， 支持对黑乐谱(black)的流畅播放  
3.可导出合成音频的mp3, wav, pcm到文件  
4.包含一个物理钢琴合成音源   
5.支持C#播放midi音乐(midi play)等  

### 支持系统平台
目前支持
   Window， 
   Android 平台
